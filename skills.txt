# Skills + agents to install. Both install-skills.ps1 and install-skills.sh read this file.
#
# Format (3 columns, whitespace-separated):
#   <category>  <kind>  <category/name>
#
#   category = core | dev | design | mobile | marketing | advanced | design-extra
#   kind     = skill | agent
#
# Default install profile: core,dev,design (typical web/SaaS project)
# Override:
#   PowerShell : .\install-skills.ps1 -Categories core,mobile
#   Bash       : ./install-skills.sh --categories all
# Special token: `all` installs every category.
#
# Why categories: each installed skill loads its description into context at session
# start. Fewer skills = less ambient context = more room for real work.
#
# Catalog: https://aitmpl.com  /  https://github.com/davila7/claude-code-templates
